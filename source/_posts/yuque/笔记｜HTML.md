### 说明

记录一些我用到的 HTML 的语法

## 跳转 URL

### 1.

```
<html>
<head>
<meta http-equiv="refresh" content="0;URL=Your Url">
</head>
</body>
</html>
```

### 2.

```javascript
<script>location.href="Your Url";</script>
```

```javascript
<script>windows.location.href = "Your Url"</script>
```

## 图片更改大小

```
<img src="img.jpg" width="150" height="150">
```
